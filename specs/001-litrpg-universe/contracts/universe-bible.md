# Contract: Universe Bible

**Deliverable**: `universe/bible/`
**Purpose**: Comprehensive reference for all world-building elements

---

## Required Documents

### 1. cosmology.md

**Content Requirements**:
- [ ] How the galactic entertainment system operates
- [ ] Physics/magic rules of the universe
- [ ] How the System was created/discovered
- [ ] Relationship between real space and dungeon spaces
- [ ] Timeline of galactic history relevant to games

**Word Count**: 2,000-4,000 words

**Validation**: Reader should understand universe rules without contradictions

---

### 2. species/ (directory)

**Required Files** (minimum 5):
- [ ] `species-1.md` - Hostile species profile
- [ ] `species-2.md` - Sympathetic species profile
- [ ] `species-3.md` - Indifferent species profile
- [ ] `species-4.md` - Fascinated species profile
- [ ] `species-5.md` - Wildcard species profile

**Per-Species Content**:
- Name and common designations
- Physical description (with visual reference notes)
- Cultural overview
- Political structure
- Relationship to games/System
- Attitude toward humans
- Notable individuals
- Faction affiliations

**Word Count**: 500-1,000 words per species

---

### 3. factions/ (directory)

**Required Files** (minimum 4):
- [ ] `faction-1.md` - Resource-focused faction
- [ ] `faction-2.md` - Sympathetic faction
- [ ] `faction-3.md` - Profit-focused faction
- [ ] `faction-4.md` - Hidden agenda faction

**Per-Faction Content**:
- Name and symbols
- Core philosophy
- Economic model
- Political stance on games
- Key species members
- Named representatives
- Relationship to protagonist
- Secret agenda (if any)

**Word Count**: 800-1,500 words per faction

---

### 4. history.md

**Content Requirements**:
- [ ] Origin of the games (how they started)
- [ ] Major historical events
- [ ] Previous notable contestants
- [ ] Earth's discovery and selection
- [ ] Recent political shifts

**Word Count**: 1,500-3,000 words

---

### 5. glossary.md

**Content Requirements**:
- [ ] Canonical terminology with definitions
- [ ] Pronunciation guides where needed
- [ ] Terms to avoid (deprecated/rejected options)
- [ ] Cross-references between entries

**Format**: Alphabetical listing with consistent entry format

---

## Acceptance Criteria

- [ ] All required documents present
- [ ] No contradictions between documents
- [ ] Terminology consistent across all files
- [ ] Constitution principles reflected (especially Principle V - Mechanics Integrity)
- [ ] Mystery seeds planted for series-long arcs
