# Universe Bible Rules

**Applies to**: `universe/bible/**/*.md`

## Document Structure Requirements

### Species Files (`universe/bible/species/*.md`)

Required sections:
- Name and common designations
- Physical description (with visual reference notes)
- Cultural overview
- Political structure
- Relationship to games/System
- Attitude toward humans (hostile/sympathetic/indifferent/fascinated/wildcard)
- Notable individuals
- Faction affiliations

Word count: 500-1,000 words per species

### Faction Files (`universe/bible/factions/*.md`)

Required sections:
- Name and symbols
- Core philosophy
- Economic model
- Political stance on games
- Key species members
- Named representatives
- Relationship to protagonist
- Secret agenda (if any)

Word count: 800-1,500 words per faction

### cosmology.md Requirements

Must cover:
- How the galactic entertainment system operates
- Physics/magic rules of the universe
- How the System was created/discovered
- Relationship between real space and dungeon spaces
- Timeline of galactic history relevant to games

Word count: 2,000-4,000 words

### history.md Requirements

Must cover:
- Origin of the games
- Major historical events
- Previous notable contestants
- Earth's discovery and selection
- Recent political shifts

Word count: 1,500-3,000 words

### glossary.md Requirements

- Alphabetical listing
- Consistent entry format
- Pronunciation guides where needed
- Terms to avoid (deprecated options)
- Cross-references between entries

## Consistency Rules

1. **No contradictions** - Check existing bible content before adding new lore
2. **Terminology locked** - Use terms from glossary.md exactly
3. **Mystery budget** - Plant seeds but track payoffs (mysteries need planned answers)
4. **Faction complexity** - No pure good/evil; all factions have comprehensible motivations

## Cross-Reference Checklist

Before finalizing bible content:
- [ ] Terms match glossary.md
- [ ] Species attitudes align with faction descriptions
- [ ] Timeline consistent with history.md
- [ ] System mechanics reference mechanics/ docs
- [ ] Constitution Principle V (Mechanics Integrity) validated
